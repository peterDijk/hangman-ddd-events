## Description

project to learn about DDD and Event Sourcing

in a Docker setup using Eventstore

## Installation

```bash
$ docker-compose build
```

## Running the app

```bash
# development
$ ./scripts/build-up.sh
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
